Plan: Agentic BoltzGen Protein Design Workflow                                                                                                                                                                                               │
     │                                                                                                                                                                                                                                              │
     │ Context                                                                                                                                                                                                                                      │
     │                                                                                                                                                                                                                                              │
     │ Build a Python-based agentic workflow repository (biohack) for protein design campaigns using BoltzGen. The repo currently contains only install.sh. The system will orchestrate iterative design rounds: run BoltzGen, analyze results, and │
     │  propose improved specifications — with the primary optimization target being high Boltz 2 ipSAE scores.                                                                                                                                     │
     │                                                                                                                                                                                                                                              │

     │ Key Data Models (src/protdesign/models/)                                                                                                                                                                                                     │
     │                                                                                                                                                                                                                                              │
     │ campaign.py                                                                                                                                                                                                                                  │
     │                                                                                                                                                                                                                                              │
     │ - CampaignConfig: Immutable target configuration — target_cif_path, protocol, binding_site_residues, binder_length_range, num_designs_per_round, budget_per_round, goal_ipsae_threshold                                                      │
     │ - RoundSummary: Persisted round stats — status, best_ipsae_min, mean_ipsae_min, pct_above_threshold, paths to agent outputs                                                                                                                  │
     │ - Campaign: Full state — campaign_id, config, list of RoundSummary, all_time_best_ipsae, strategy_notes (cross-round agent memory)                                                                                                           │
     │                                                                                                                                                                                                                                              │
     │ design.py                                                                                                                                                                                                                                    │
     │                                                                                                                                                                                                                                              │
     │ - DesignMetrics: Per-design metrics from CSV — design_ptm, design_iptm, design_ipsae_min, min_design_to_target_pae, delta_sasa, plip_hbonds, plip_saltbridge                                                                                 │
     │ - RoundDesigns: All designs + final designs for a round, with best_by_ipsae() and summary_stats() methods                                                                                                                                    │
     │                                                                                                                                                                                                                                              │
     │ spec.py                                                                                                                                                                                                                                      │
     │                                                                                                                                                                                                                                              │
     │ - BoltzGenSpec: Mirrors BoltzGen YAML structure (entities + constraints), with to_yaml_file(path) method                                                                                                                                     │
     │                                                                                                                                                                                                                                              │
     │ ---                                                                                                                                                                                                                                          │
     │ BoltzGen Tool (src/protdesign/tools/boltzgen_runner.py)                                                                                                                                                                                      │
     │                                                                                                                                                                                                                                              │
     │ Subprocess wrapper around boltzgen run:                                                                                                                                                                                                      │
     │                                                                                                                                                                                                                                              │
     │ @dataclass                                                                                                                                                                                                                                   │
     │ class BoltzGenRunConfig:                                                                                                                                                                                                                     │
     │     spec_yaml_path: Path                                                                                                                                                                                                                     │
     │     output_dir: Path                                                                                                                                                                                                                         │
     │     protocol: str = "protein-anything"                                                                                                                                                                                                       │
     │     num_designs: int = 1000                                                                                                                                                                                                                  │
     │     budget: int = 20                                                                                                                                                                                                                         │
     │     steps: list[str] | None = None   # None = all steps                                                                                                                                                                                      │
     │     reuse: bool = False                                                                                                                                                                                                                      │
     │     cache_dir: Path | None = None                                                                                                                                                                                                            │
     │                                                                                                                                                                                                                                              │
     │ def run_boltzgen(config: BoltzGenRunConfig) -> BoltzGenRunResult:                                                                                                                                                                            │
     │     # Build CLI command, stream stdout/stderr live, return result                                                                                                                                                                            │
     │                                                                                                                                                                                                                                              │
     │ Command built as: boltzgen run <spec.yaml> --output <dir> --protocol <p> --num_designs <n> --budget <b> [--reuse] [--steps ...]                                                                                                              │
     │                                                                                                                                                                                                                                              │
     │ ---                                                                                                                                                                                                                                          │
     │ Metrics Parser (src/protdesign/tools/metrics_parser.py)                                                                                                                                                                                      │
     │                                                                                                                                                                                                                                              │
     │ def parse_round_output(boltzgen_output_dir: Path, round_number: int) -> RoundDesigns:                                                                                                                                                        │
     │     # Reads: final_ranked_designs/all_designs_metrics.csv                                                                                                                                                                                    │
     │     #        final_ranked_designs/final_designs_metrics_<N>.csv                                                                                                                                                                              │
     │     # Falls back to: intermediate_designs_inverse_folded/aggregate_metrics_analyze.csv                                                                                                                                                       │
     │                                                                                                                                                                                                                                              │
     │ ipSAE is typically included in BoltzGen's output CSVs as design_ipsae_min/design_to_target_ipsae. If absent, compute from .npz files in fold_out_npz/ using raw PAE matrices.                                                                │
     │                                                                                                                                                                                                                                              │
     │ ---                                                                                                                                                                                                                                          │
     │ Agents (src/protdesign/agents/)                                                                                                                                                                                                              │
     │                                                                                                                                                                                                                                              │
     │ Base Agent (base_agent.py)                                                                                                                                                                                                                   │
     │                                                                                                                                                                                                                                              │
     │ Standard Claude API agentic loop:                                                                                                                                                                                                            │
     │ - Takes a system prompt, user message, and list of tools                                                                                                                                                                                     │
     │ - Calls anthropic.Anthropic().messages.create() with tools=                                                                                                                                                                                  │
     │ - Handles tool_use content blocks — dispatches to registered tool functions                                                                                                                                                                  │
     │ - Loops until stop_reason == "end_turn"                                                                                                                                                                                                      │
     │ - Model: claude-sonnet-4-6                                                                                                                                                                                                                   │
     │                                                                                                                                                                                                                                              │
     │ Monitor Agent (monitor_agent.py)                                                                                                                                                                                                             │
     │                                                                                                                                                                                                                                              │
     │ Input: Campaign state + RoundDesigns for the latest round                                                                                                                                                                                    │
     │ Tools available to agent:                                                                                                                                                                                                                    │
     │ - get_round_metrics(round_number) → returns RoundDesigns summary stats                                                                                                                                                                       │
     │ - get_top_designs(round_number, n, sort_by) → returns top-N designs with metrics                                                                                                                                                             │
     │ - compare_rounds() → trend analysis across all rounds                                                                                                                                                                                        │
     │ - get_design_sequence(design_id) → returns sequence string                                                                                                                                                                                   │
     │ - get_campaign_config() → returns CampaignConfig                                                                                                                                                                                             │
     │                                                                                                                                                                                                                                              │
     │ Output: Structured MonitorReport JSON with:                                                                                                                                                                                                  │
     │ - assessment: overall campaign health (one of: on_track, plateau, poor_convergence, goal_reached)                                                                                                                                            │
     │ - key_findings: list of observations (e.g., "ipSAE improving +0.05/round", "designs clustering in 80aa range")                                                                                                                               │
     │ - top_design_ids: list of best design IDs to carry forward                                                                                                                                                                                   │
     │ - bottlenecks: list of identified issues                                                                                                                                                                                                     │
     │ - recommended_focus: high-level guidance for designer                                                                                                                                                                                        │
     │                                                                                                                                                                                                                                              │
     │ Designer Agent (designer_agent.py)                                                                                                                                                                                                           │
     │                                                                                                                                                                                                                                              │
     │ Input: MonitorReport + full Campaign state (including strategy_notes)                                                                                                                                                                        │
     │ Tools available to agent:                                                                                                                                                                                                                    │
     │ - get_current_spec(round_number) → returns YAML spec for a previous round                                                                                                                                                                    │
     │ - get_campaign_config() → returns base configuration                                                                                                                                                                                         │
     │ - validate_spec(spec_yaml_str) → validates BoltzGen YAML syntax                                                                                                                                                                              │
     │ - get_boltzgen_protocol_info(protocol) → returns protocol documentation                                                                                                                                                                      │
     │                                                                                                                                                                                                                                              │
     │ Output: Structured DesignerProposal JSON with:                                                                                                                                                                                               │
     │ - spec: complete BoltzGenSpec (serialized to YAML for next round)                                                                                                                                                                            │
     │ - num_designs: suggested --num_designs value                                                                                                                                                                                                 │
     │ - budget: suggested --budget value                                                                                                                                                                                                           │
     │ - rationale: explanation of changes from previous round                                                                                                                                                                                      │
     │ - strategy_note: a sentence to append to campaign.strategy_notes                                                                                                                                                                             │
     │                                                                                                                                                                                                                                              │
     │ ---                                                                                                                                                                                                                                          │
     │ Orchestrator (src/protdesign/orchestrator.py)                                                                                                                                                                                                │
     │                                                                                                                                                                                                                                              │
     │ class CampaignOrchestrator:                                                                                                                                                                                                                  │
     │     def run_next_round(self, campaign_id: str) -> RoundSummary:                                                                                                                                                                              │
     │         """                                                                                                                                                                                                                                  │
     │         Full cycle:                                                                                                                                                                                                                          │
     │         1. Load campaign state                                                                                                                                                                                                               │
     │         2. If round 0: build initial spec from CampaignConfig                                                                                                                                                                                │
     │            Else: run DesignerAgent to get new spec                                                                                                                                                                                           │
     │         (For round 0: build minimal spec from CampaignConfig — just target chain + binder length range, use BoltzGen defaults for everything else)                                                                                           │
     │         3. Run BoltzGen (boltzgen_runner)                                                                                                                                                                                                    │
     │         4. Parse outputs (metrics_parser)                                                                                                                                                                                                    │
     │         5. Run MonitorAgent → save monitor_report.json                                                                                                                                                                                       │
     │         6. Update campaign state with RoundSummary                                                                                                                                                                                           │
     │         7. Save campaign.json                                                                                                                                                                                                                │
     │         8. Return RoundSummary                                                                                                                                                                                                               │
     │         """                                                                                                                                                                                                                                  │
     │                                                                                                                                                                                                                                              │
     │     def run_campaign(self, campaign_id: str, max_rounds: int | None = None) -> Campaign:                                                                                                                                                     │
     │         """Loop run_next_round until goal reached or max_rounds hit."""                                                                                                                                                                      │
     │                                                                                                                                                                                                                                              │
     │ ---                                                                                                                                                                                                                                          │
     │ CLI (src/protdesign/cli.py)                                                                                                                                                                                                                  │
     │                                                                                                                                                                                                                                              │
     │ protdesign campaign create --target target.cif --name my_campaign [options]                                                                                                                                                                  │
     │ protdesign campaign list                                                                                                                                                                                                                     │
     │ protdesign campaign status <campaign_id>                                                                                                                                                                                                     │
     │ protdesign run <campaign_id>              # run one round, then pause for review                                                                                                                                                             │
     │ protdesign run <campaign_id> --auto       # loop until goal/max_rounds (no pauses)                                                                                                                                                           │
     │ protdesign report <campaign_id>           # print rich summary table                                                                                                                                                                         │
     │                                                                                                                                                                                                                                              │
     │ Default (human checkpoint) mode: after each round, print the monitor report + proposed YAML spec to stdout, then prompt [A]pprove / [E]dit / [Q]uit. If "edit", open spec in $EDITOR. If "approve", proceed.                                 │
     │                                                                                                                                                                                                                                              │
     │ --auto mode: fully autonomous loop, no prompts.                                                                                                                                                                                              │
